# Browser Extension Integration Checklist

## Chromium

- Install extension zip from Release artifacts
- Verify local bridge endpoint and token paste flow
- Send current page URL from popup
- Send sniffed media candidate
- Verify token mismatch error message

## Firefox

- Load Firefox build from Release artifacts
- Verify local bridge endpoint and token paste flow
- Send current page URL from popup
- Send sniffed media candidate
- Verify origin allowlist behavior
